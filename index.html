Guide

작성 규격 (Method)

완성본은 표준 HTML 문서가 아니라 호스트 페이지 안에 삽입되는 본문 조각입니다.

  • 출력은 <style>@import…</style> + <main class="page">로 시작해 </main>으로 끝낸다.
  • <!DOCTYPE>·<html>·<head>·<body>·<script> 금지. CSS만 @import로 로드.
  • 좁은 단일 칼럼(640px). 섹션 헤더는 .sec--bar 또는 .sec--quote.
  • 전체 규격은 저장소 CLAUDE.md 참고.
<style type="text/css">@import url("./style.css");</style>
<main class="page">
  <article> … </article>
</main>
